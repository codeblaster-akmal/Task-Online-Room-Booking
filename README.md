### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/codeblaster-akmal/Task-Online-Room-Booking.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your SERVER & DB Info in `.env`
   ```JS
  PORT=<<3000>>
  JWT_SECRET_KEY=<<jwtsecretkey>>
  
  DATABASE_URI=<<database_uri>>
   ```
4. Run project
   ```sh
   npm start
   ```
